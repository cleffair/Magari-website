# Magari Studio Website Rules

Tech Stack

* Next.js 15
* TypeScript
* Tailwind CSS
* shadcn/ui
* Motion

Design Principles

* Editorial luxury fashion aesthetic
* Mobile first
* Minimal
* Spacious
* Photography driven
* Slow fashion storytelling

References

* Nicobar
* Good Earth
* Toteme
* COS

Avoid

* Generic Shopify appearance
* Bootstrap
* Material UI
* Heavy gradients
* Glassmorphism
* Excessive animations

Code Standards

* Reusable components
* Server components where possible
* Semantic HTML
* Accessibility support
* Optimized images

Content Source

* Instagram photography
* Studio photography
* Founder story
* Google Maps location

Primary Goal

Present Magari Studio as a premium contemporary Indian fashion label and generate Instagram and WhatsApp inquiries.
